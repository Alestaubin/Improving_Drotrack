Improving DroTrack: Insights and Advancements for Drone-based Tracking

Alexandre St-Aubin, Alexis Viau-Cardinal, Rafid Saif

Code

Abstract

In this work, we present advancements to the DroTrack algorithm for improved object tracking in drone-based applications. Our proposed method enhances accuracy and robustness in long-range, real-time scenarios. We compare DroTrack to our improvements using multiple examples, highlighting improvements.
